# NKUST 成績監控小幫手 (NKUST Grade Monitor) 🎓
![Downloads](https://img.shields.io/github/downloads/xixa3333/Score_Monitor_Release/total)

這是一個自動化的成績查詢工具，專為高科大 (NKUST) 學生設計。它會自動登入校務系統，持續監控成績頁面，一旦發現新成績公布或更新，就會自動寄 Email 通知你！

## ✨ 功能特色
- **全自動監控**：自動登入、自動刷新，無需手動操作。
- **斷線重連**：若被系統登出或發生錯誤，程式會自動重啟瀏覽器。
- **即時通知**：偵測到新成績後，立即發送 Email。
- **貼心評語**：依據分數自動判斷是否「上岸」(Pass) 並附上趣味評語（老師撈不動 vs 撈上岸）。
- **錯誤修復**：自動處理 ASP.NET 錯誤頁面與驗證彈窗。

## 🚀 如何使用 (Windows)

1. 前往 [Releases](https://github.com/xixa3333/Score_Monitor_Release/releases) 下載最新的 `NKUST_Score_Monitor.zip`。
2. 解壓縮檔案，確保 `.exe` 與 `config.txt` 在同一資料夾。
3. 打開 `config.txt`，填入你的：
   - 學號 (Student_ID)
   - 密碼 (Student_Password)
   - 接收通知的信箱 (Target_Email)
   - 取得應用程式密碼設定Email通知，底下有設定方法
4. 點擊執行 `.exe` 檔案。
5. 視窗跳出後請勿關閉，程式將在背景持續運作。

## 📧 如何設定 Email 通知？ (v3.0 必讀)
為了確保穩定性與隱私，本程式需要您設定自己的 Gmail 寄件服務：

1. 前往您的 [Google 帳號安全性設定](https://myaccount.google.com/security)。
2. 開啟 **「兩步驟驗證」**。
3. 搜尋並進入 **「應用程式密碼」** (App Password)。
4. 建立一個名稱（例如：`NKUST-Score`），系統會給您一串 **16 位數的密碼**。
5. 將此密碼填入 `config.txt` 中的 `App_Password` 欄位即可。

## 💡 進階技巧：如何讓程式開機自動執行？

如果你希望電腦一開機，程式就自動跳出來幫你監控成績，請依照以下步驟設定：

1. **建立捷徑**：
   - 對著 `NKUST_Score_Monitor.exe` 檔案按滑鼠右鍵，選擇 **「建立捷徑」**。
2. **開啟啟動資料夾**：
   - 按下鍵盤上的 **`Win`** + **`R`** 鍵（打開執行視窗）。
   - 在框框內輸入 `shell:startup`，然後按下 **「確定」**。
   - 此時會跳出一個資料夾視窗。
3. **放入捷徑**：
   - 將剛剛建立的 **捷徑** 拖曳（或複製貼上）到這個打開的資料夾內。

**完成！** 🎉
下次當你重新啟動電腦並登入 Windows 時，這個程式就會自動啟動並開始監控成績囉！

## 📝 注意事項
- 程式執行時會產生 `system_log.txt`，若發生問題可查看此檔案。
- 首次執行會建立 `grade_history.json` 用於記錄已通知的成績。
- 請勿關閉跳出的黑色終端機視窗，否則程式會停止。
